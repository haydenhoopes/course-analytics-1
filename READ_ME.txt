This Flask Project was commissioned by the Center for Student Analytics.
November 14, 2019

The enclosed application allows authorized faculty to generate non-exportable
tables with information about courses and their assignment quantity and types. It also
allows faculty to see summaries of default views for courses in a particular department.

INSTRUCTIONS FOR USE:
1.  Make sure the most current version of Python is installed.
2.  Click the file packages.run
3.  Enter your canvas token.

If you have any questions, contact the Center for Student Analytics
ais.usu.edu/analytics
